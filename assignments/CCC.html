Steps of the Computer Controlled Cutting

  1. Design: Draft vectors (e.g., in Fusion 360, Illustrator, or Inkscape). Use proper dimensions and join paths.
  2. Material Selection: Choose sheet material (acrylic, plywood, cardboard) and confirm thickness/tolerances.
  3. Prepare the File: Export to machine-ready format (e.g., SVG/DXF). Color-code cuts/engraves if needed.
  4. Machine Setup: Focus the blade, set origin/home, secure material, check ventilation/safety.
  5. Power & Speed: Set parameters for cut vs. score/engrave. Run a small test cut for kerf/fit.
  6. Cutting: Send the job, monitor progress, pause if flames/warping occur.
  7. Post-Processing: Remove parts, clean edges, test fit, and iterate on the design as needed.
